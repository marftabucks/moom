Moom - Mall in Mobile
(Front End)
--------------------------
Moom is built using React Native to reach both android and iOS users

Technologies:
- React Native
- Expo
- Third party libraries (React navigation, ImageViewer)
--------------------------

Folder structures:
- Assets: Images, Icons, and Placeholders
- Components: Reusable react components to maintain modularity
- Constants: Reusable and centralized Constants
- Fonts: Fonts that are used in this project
- Navigations: Navigations to navigate across the screens
- Screens: Screens that appears in this project
